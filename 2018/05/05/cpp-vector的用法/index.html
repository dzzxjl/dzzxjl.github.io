vector容器的方法

  • int size() : 返回容器内的元素个数
  • void clear() ：清空所有元素
  • bool empty() ：判断是否为空
  • void push_back(int x) ：在表尾添加x
  • void pop_back() ：删除表尾元素
  • back() ：返回表尾元素
  • front() ：返回表头元素
  • operator[] (int idx) ：返回vector中idx索引位置的对象
  • int capacity() ：返回vector的内部容量
  • void reserve(int capacity) ：设置vector的新容量
