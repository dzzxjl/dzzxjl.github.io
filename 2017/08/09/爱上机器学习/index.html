爱上机器学习

2017-08-09

什么是机器学习？

机器学习指的是机器通过统计学算法，对大量的历史数据进行学习从而生成经验模型，利用经验模型指导业务。

  • 输入：数据 + 算法
  • 输出：模型

SAMPLE

挖掘目标：例如挖掘流、时序和序列数据以及挖掘时间空间、多媒体（音频speech、图像image、视频等）、文本、Web数据处理

机器学习分类

半监督学习（弱监督学习）

有监督学习

  • 我们已经知道预测所应该得出的正确答案，训练集能起到监督和指导的作用，在神经网络预测错误时予以纠正，从而优化模型
  • 分类模型（回归模型、KNN、决策树DT、SVM、神经网络）
  • 预测模型

无监督学习

  • 不告诉你结果或标准答案，让你从已知的数据中找到哪些特征
  • 关联分析
  • 聚类分析（k-means聚类、Birch聚类、密度聚类、层次聚类）

经典书籍

  • 李航《统计学习方法》
  • 周志华《机器学习》
  • 宗成庆《统计自然语言处理》

数学模型

  • 隐马尔科夫模型 + 最大熵模型 = 条件随机场（CRF）（在分词、词性标注和命名实体识别等序列标注任务中取得了很好的效果）
  • SVM（支持向量机）是一个有监督的学习模型，通常用来进行模式识别、分类以及回归分析。SVM（支撑向量机）是数学规划方法在分类上的一个典型应用。SVM学习问题可以表示为凸优化问题，因此可以利用已知的有效算法发现目标函数的全局最小值。而其他分类方法（如基于规则的分类器和人工神经网络）都采用一种基于贪心学习的策略来搜索假设空间，这种方法一般只能获得局部最优解。SVR是支持向量回归(support vector regression)的英文缩写，是支持向量机(SVM)的重要的应用分支。
  • 随机森林（RF）
  • 决策树（GBDT(梯度提升决策树)决策模型，分类方法）
  • 贝叶斯

回归问题（逻辑回归、线性回归）

拟合：模型（构建的函数）与真实数据之间的贴合程度

欠拟合：对深度学习而言，就是训练得不够，数据不足

过拟合：过拟合是指模型对定型数据的拟合非常好，然而一旦在实际应用中遇到从未出现过的数据，运行效果就变得很不理想。
