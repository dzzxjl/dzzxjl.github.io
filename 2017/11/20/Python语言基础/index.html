在命令行中添加以上参数，可以让pip从指定的镜像源安装软件

永久修改：

  1. 进入根目录：cd ~/
  2. 进入.pip目录 cd .pip
  3. 如果不存在文件夹就新建mkdir .pip
  4. 进入 cd .pip
  5. 创建pip.conf文件 touch pip.conf
  6. 修改：vim pip.conf

目前收集的比较好的镜像地址有：

  • http://mirrors.aliyun.com/pypi/simple/
